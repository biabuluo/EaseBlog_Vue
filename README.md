# EaseBlog_Vue
*EaseBlog前端项目*





## 项目所需依赖

> - axios : 请求
>
> - pinia ：状态管理
>
> - sass ：简化css
>
> - vue-router ： 路由
>
>   `npm install vue-router@4`
>
> - naive-ui ：ui框架
>
>    `npm i -D naive-ui`
>
>   `npm i -D vfonts`
>
> - wangeditor ： 富文本
>
>   `npm  install @wangeditor/editor-for-vue@next --save`
